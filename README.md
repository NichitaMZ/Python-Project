# Customer Management System (Python + MySQL)

## Description
This project is a command-line based customer and sales management system developed using Python and MySQL. It allows users to manage customers, products, suppliers, and sales records efficiently.

## Features
- Add, update, and delete customer data
- Manage product and supplier information
- Record and track sales
- Generate reports (PDF/Excel)
- Improved security using .env and .gitignore

## Technologies Used
- Python
- MySQL

## Project Structure
- main_menu.py – Main interface
- customer_table.py – Customer operations
- product_table.py – Product management
- sales_table.py – Sales handling
- suppliers_table.py – Supplier data
- report_sale.py – Sales reporting

## How to Run
1. Install Python
2. Set up MySQL database
3. Connect Python with Mysql
4. Update database credentials in the code
5. Run:
   python main_menu.py
